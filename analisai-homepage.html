Fitur Unggulan

Perangkat lengkap
untuk tim data

Dari eksplorasi data hingga deployment model — semua dalam satu pipeline terintegrasi.

01 ——
Planner + Executor Agent
Pipeline 3-agent berbasis LangGraph: Planner menyusun rencana, Executor menjalankan kode paralel per fase, Critic mengevaluasi hasil secara otomatis.
02 ——
Sandbox Docker Terisolasi
Seluruh eksekusi kode Python berjalan dalam container Docker offline. Data Anda tidak pernah meninggalkan environment terisolasi.
03 ——
AutoML Terstruktur
5 model candidates (RF, GBM, XGB, LGBM, Linear) dengan cross-validation, hyperparameter tuning otomatis, dan leaderboard performa.
04 ——
Streaming Job Tangguh
Proses AI berjalan di background via Redis queue. Refresh halaman kapanpun — progres tidak hilang. Event streaming real-time ke frontend.
05 ——
Dashboard Streamlit Otomatis
AI menghasilkan aplikasi web Streamlit interaktif secara otomatis dari instruksi teks. Berjalan di container terpisah tanpa konfigurasi manual.
06 ——
Multi-format & MinIO Storage
Dukungan CSV, Excel, JSON, Parquet, Pickle. File disimpan di MinIO object storage — aman, persisten, dan dapat diakses lintas sesi.
Cara Kerja

Dari teks ke insight
dalam detik

Pipeline otomatis yang mengubah pertanyaan bahasa natural menjadi analisis data yang komprehensif.

01.
Unggah & Tanya
Upload file data (CSV/Excel/JSON) dan ketik pertanyaan dalam bahasa Indonesia. Tidak perlu syntax SQL atau kode apapun.
02.
Planner Agent Menyusun Rencana
LLM planner menganalisis pertanyaan dan membuat JSON task plan dengan fase paralel — seperti sprint planning tapi dalam milidetik.
03.
Executor Menjalankan Kode
Kode Python dieksekusi secara paralel per fase di sandbox Docker terisolasi. Grafik, model, dan output dikirim real-time ke browser Anda.
04.
Critic Agent Mengevaluasi
Critic memeriksa kelengkapan hasil. Jika kurang, ia membuat additional tasks untuk refinement otomatis — tanpa intervensi manual.

Arsitektur

Pipeline yang
dirancang untuk skala

Setiap komponen terisolasi, scalable, dan dapat dikonfigurasi secara independen.

input layer
Classifier & Router
Lightweight LLM classifier menentukan routing: smalltalk, model_build, simple task, atau full 3-agent pipeline.
classifier.py
planning layer
Planner Agent
MODEL_PLANNER menghasilkan JSON task plan dengan fase paralel. Fallback otomatis jika LLM gagal — pipeline tidak pernah stuck.
pipeline.py
execution layer
Executor + Sandbox
Tasks dijalankan paralel per fase via threading. Python sandbox Docker terisolasi dengan timeout 300s per task.
executor.py sandbox.py
evaluation layer
Critic Agent
Evaluasi output dengan threshold metrik. Jika performa rendah, Critic menghasilkan additional tasks untuk refinement otomatis.
